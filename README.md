Sistema de Países 🌎

Este projeto é um sistema simples de informações sobre países, desenvolvido em Python. Ele permite consultar informações como moedas, população e o número total de países.

O projeto foi criado durante minhas aulas na Solyd, enquanto eu aprendia programação e manipulação de dados de APIs.

Funcionalidades

Contar o número total de países.

Listar a moeda de um país específico.

Mostrar a população de um país específico.

Como usar

Execute pelo terminal com o comando:

python paises.py <ação> <nome do país>
Ações disponíveis

contagem — exibe o número total de países.

moeda — exibe as moedas do país indicado.

populacao — exibe a população do país indicado.

Exemplos:

python paises.py contagem

python paises.py moeda Brasil

python paises.py populacao Alemanha
